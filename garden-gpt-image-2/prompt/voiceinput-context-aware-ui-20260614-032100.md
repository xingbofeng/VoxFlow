# 随声写 VoxFlow 上下文工作流 UI 设计稿

生成模式：Host-Native GPT Image 2

共同视觉约束：

- 高保真原生 macOS SwiftUI 工具界面，完整桌面窗口。
- 延续随声写 VoxFlow 现有三栏结构、浅灰绿侧栏、鼠尾草选中态和紧凑信息密度。
- 中文系统字体，小圆角，薄分隔线，不使用网页 SaaS、渐变、玻璃拟态或营销 Hero。
- 使用语义准确、清晰可辨的应用图标，不使用字母占位。
- 画布为 1536 x 1024 横向界面稿。

## 1. 设置页：帮我说快捷键

在现有“设置 → 通用”页中保留输入设备卡片，将快捷键区域改为两个动作：

- “语音转录”：按住快捷键说话，松开后转写并输入；当前为右 Command；提供“修改”。
- “帮我说”：结合当前窗口和口述生成文本，完成后复制到剪贴板；未设置时提供主按钮“设置快捷键”。
- 显示“不自动发送”边界提示。
- 提供“按住 / 切换”触发方式，并说明两个动作不能使用相同触发方式。

## 2. 风格页：适用应用

保留现有风格列表和 Markdown / 预览双栏，在右侧编辑区标题下新增紧凑的“适用应用”区域：

- 展示微信、飞书、Slack、信息等真实应用图标和名称。
- 显示“4 个应用”。
- 提供“管理应用”添加入口和“智能配置”按钮。
- 应用区域高度克制，不挤压 Prompt 编辑器。

## 3. 管理应用弹窗

在风格页上方显示原生居中模态框：

- 标题“管理应用”。
- “已选应用 (8)”区域使用应用图标、名称和移除按钮，展示 Cursor、Visual Studio Code、Xcode、IntelliJ IDEA、PyCharm、WebStorm、Terminal、iTerm。
- “从已安装应用添加”包含搜索框。
- 列表展示应用图标、名称、Bundle ID 和添加按钮。
- 底部提供“浏览… / 重置 / 取消 / 保存”。

## 4. 智能配置应用

在随声写 VoxFlow 风格页中展示确认工作流：

- 标题“智能配置应用”，说明已扫描 18 个应用。
- 按“编程、聊天、邮件、通用”分组。
- 使用真实应用图标。
- 每个应用显示“系统预设”或“AI 推荐”来源标签，并提供移动分类和移除操作。
- 页脚说明“仅发送应用名称、Bundle ID 和系统分类，不读取应用内容。”
- 底部提供“暂不应用”和“应用配置”。
